97 Percent is an application that enables a user to find their representative in Washington D.C. by entering their home address. When the user enters their address, a call is made to the Google Civics API which then returns the information requested.

Wilkipedia
WiLkipedia is a WikiPedia viewer that a user can use to lookup articles on Wikipedia using the portal that I developed. It works by taking the entered string and returning the top ten matches to what was entered. Users can then link directly to the article.

This is a weather application that was developed to the specifications in the FreeCodeCamp ciriculum. It works by getting the users current location via the browser, and returns the temperature, condition, time, and also shows the location of the user.
