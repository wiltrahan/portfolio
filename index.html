RECENT WORK

click on image for github repo

97% is an application that enables a user to find their representative in Washington D.C. by entering their home address. When the user enters their address, a call is made to the Google Civics API which then returns the information requested.

WiLkipedia is a WikiPedia viewer that a user can use to lookup articles on Wikipedia using the portal that I developed. It works by taking the entered string and returning the top ten matches to what was entered. Users can then link directly to the article.

This is a weather application that was developed to the specifications in the FreeCodeCamp ciriculum. It works by getting the users current location via the browser, and returns the temperature, condition, time, and the location of the user.

FlickList is a class project that I completed during my time in LaunchCode's LC101 program. It lets users keep a running "to-do" list of movies they've been meaning to watch and provides an interface for browsing new movies, using the moviedb API.
